# Influenza Vaccination Coverage Dashboard

This Tableau dashboard visualizes influenza vaccination coverage across U.S. states
and compares state-level coverage against the Healthy People 70% vaccination target.

## Dashboard Contents
- U.S. map showing influenza vaccination coverage by state
- Time-series trends of vaccination coverage by year
- Comparison of state coverage vs 70% national target
- Identification of underperforming states

## Data Source
- CDC FluVaxView: Influenza Vaccination Coverage (All Ages, 6+ Months)

## Notes
- The dashboard is packaged as a `.twbx` file and must be opened in Tableau Desktop.
- Due to GitHub preview limitations, the dashboard cannot be rendered directly in the browser.
- The dashboard connects to a SQL-generated analytical table created in the `scripts/` folder.

## How to View
1. Download the `.twbx` file
2. Open it using Tableau Desktop
3. Review interactive filters and state-level comparisons
